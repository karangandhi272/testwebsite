A functions is simpy a relationship between one varible with another. In most cases these varibles include x and y. The x is manipulatied to result in a y. The only rule with a function is one x(in this case) varible cannot result in two y varible. However two different x varibles can lead to one y varible and that is completely fine.

This is a functions as there is only 1 y for every x (y = x + 1)

This is a functions as there is only 1 y for every x (y = x^2)

This is NOT a function as there can be 2 y numbers for one x (x^2 + y^2 = 25)

A easy representation to dectect a function is simply creating a vertical line (vertical line test). If the line anywhere connects to two point it is NOT a function.

A table: Not a function as there are two different y values for one x value (The 1 has two different y values, 5 and 6)

A set: A function as there are only 1 y value for each x value
